
# Подрядок деуствий
0. Клонируйте репозиторий и сделайте форк

1. Прочитайте
res://docs/SUMMARY.md

2. Затем прочитайте
res://docs/GUIDING.md
чтобы понять, почему были приняти те или иные решения

3. Откройти вкладку плагина TODO для поиска тех задач, что требуется завершить.
Или воспользуйтесь глобальным поиском (Ctrl+Shoft+F)

4. Загрузите последние изменения из master-ветки

5. Создайте Pull request

6. Дождитесь одобрения и слияния

Комменарии к коду и документация - ваши друзья
