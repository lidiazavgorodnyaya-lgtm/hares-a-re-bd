# Обзор
Краткий экскурс в проект

## Цветовая дифференциация штанов
Красный - активы (assets) для игры, не сцены и скрипты
 Подпапки в отдельных сценах с такими активами тоже перекрашивайте в красный

Фиолетовый - документация, gdd

Розовый - примеры, не относящиеся к проекту, в релизный билд не будут включены

Зеленый - папка для POT и CSV файлов переводов на другие языки

Серый - технические части, аддоны и автозагрузки

Бирюзовый - ресурсы (res, tres, кастомные ресурсы (.gd скрипты с extends Resource))

Остальные папки без маркировки


## Структура проекта
res://addons/
Дополнения

res://assets/
Ресурсы, разделенные между проектом

res://autoloads/
Синглтоны проекта (Проект `->` Настроки проекта `->` Глобальные)

res://docs/
Хранит информацию о проекте и мыслях, позволяющих понять, почему было принято то или иное решение

res://autoloads/game_state_manager/
Менеджер, управляющие конечным автоматом
